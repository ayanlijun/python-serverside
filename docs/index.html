Overview

This is a serverside toolkit for writting python backend logic much faster!

Features

  • Lot's of GraphQL helpers, built over Ariadne
  • Authentication helpers
  • Custom Django Fields
  • Helpful generations, such as identicons
  • Custom colour loggers

Installation

{% highlight bash %} $ pip3 install serverside {% endhighlight %}
